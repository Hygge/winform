## 餐饮管理系统

> 针对饭店管理系统，用户分为经理和店员，模块有店员管理、会员管理、餐桌管理、菜单管理、结账管理

数据库：sqlite 在dal模块下db文件夹下，**使用前需要修改UI模块的App.config数据库路径配置**。
